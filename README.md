# 🛡️ ScamShield AI

ScamShield AI is a smart AI-powered scam detection system that detects phishing messages, fake links, and fraud attempts in real time.

It analyzes SMS, emails, WhatsApp messages, and URLs to warn users about potential scams.

---

## ⚡ Features
- 🔍 Detects scam keywords (OTP, bank, verify, etc.)
- 🌐 URL & fake domain detection
- 🧠 AI-based scam explanation (Gemini/Groq API)
- 📊 Risk scoring system (Low / Medium / High)
- 🎨 Interactive UI with real-time results

---

## 🛠️ Tech Stack
- HTML, CSS, JavaScript (Vite)
- AI API (Gemini / Groq)
- Vercel (Deployment)
- GitHub (Version Control)
